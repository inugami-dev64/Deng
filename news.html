Fixed windows support on DENG

Windows support was fixed in DENG. Now it is possible to use CMake in order to create Visual Studio solutions for DENG. Additional optimisations were also made to key and mouse input event system.

Full list of fixes and changes that were done are following:

  • WINDOW instance creation no longer allocates on heap memory
  • Create CMake script for windows builds
  • Fix first person camera white screen bug
  • DENG no longer uses GCC non-standard C++ language features
