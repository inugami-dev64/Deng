About

DENG(the Dimentional ENGine)

DENG is a lightweight, free and open source game engine designed for developing 2D and 3D games. It uses mainly Vulkan for rendering as well as OpenGL in the future. Right now DENG is in early stage development, which means that no editors are available, however a custom GUI system will soon be implemented, which means that the next goal of the development is to implement map editor.

Currently supported features

  • Antialiasing (MSAA)
  • 3D model loading
  • Font rendering
  • Texture mapping
  • 2D sprite generation
  • Basic UI event system
  • Perspective first person camera system
  • Editor camera system

Features yet to be implemented

  • Fully featured GUI for DENG
  • Map editor
  • Asset animator
  • Scene editor
  • Lighting
  • Shadows
  • Editor camera system
  • Modular OpenGL support
  • Wavefront OBJ parser
  • PNG and JPEG texture image loading
  • Support for animations in DENG asset file format

Goals and philosophy

The main goal of the DENG project is to create a free and open-source unbloated platform for game development, while still giving the similar functionality as alternative game engines. Our philosophy is about giving the game developer full control of everything under the hood, while keeping everything relatively simple. This means that the game developer should be able to customise the way renderer works, optimise important mathematical calculations, compile the project into an editor mode when needed etc.
